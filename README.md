## 1.安装(命令有点长, 记得复制完哦)
```
sudo rm -r -f WeChatICU-ForMac && git clone --depth=1 https://github.com/MustangYM/WeChatICU-ForMac.git && cd WeChatICU-ForMac/WeChatICU && ./install.sh
```

## 2.卸载
